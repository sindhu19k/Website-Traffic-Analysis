# Website Traffic Analysis

## Overview

This project analyzes user interaction data to understand behavior and improve conversions for Alfido Tech.

## Dataset

Includes event-level data such as **pageview, preview, click**, along with date, location, and content details.

## Key Metrics

* Users (unique linkid)
* Sessions (30-minute inactivity rule)
* Bounce Rate (single-event sessions)
* Average Session Duration

## Insights

* High drop from pageview to click
* Many users leave after one interaction
* Some content drives higher engagement

## Recommendations

* Improve call-to-action
* Promote top-performing content
* Enhance preview experience
* Target high-engagement regions
* Reduce bounce rate

Open the notebook and run all cells.
